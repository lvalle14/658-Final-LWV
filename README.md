# 658-Final-LWV
This final project for a information visualization course creates three maps that aim to create a baseline understanding of Latin American development as they relate to climate and socioeconomic pressures.
### Climate Vulnerability.ipynb  
  * Analyzes Water Stress from ECLAC CEPAL GEO Data 2000-2020
### Scoioecon.ipynb       
  * Analyzes Agriculure as a percent of a country's GDP and Access to Electricity from ECLAC CEPAL GEO Data 2000-2020 (incomplete coverage of dates)
  * Errors in code and missing code
### Risk and Development.ipynb 
  * Synthesizes the data from two previous notebooks to class Latin American conuntries into distinct profiles that indicate preparedness for climate change as it relates to development metrics
